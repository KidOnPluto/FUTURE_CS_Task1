##  Task Overview

This project presents a **passive network security assessment** conducted on the publicly available host:

**scanme.nmap.org**

The objective was to identify:

* Open ports
* Running services
* Service version information
* Basic security misconfigurations

---

## Tools Used

* Nmap (Port and Service Scanning)
* Browser Developer Tools (Cookies Analysis)
* Microsoft Excel (Charts)

---

##  Findings Summary

* Open ports detected (SSH, HTTP, etc.)
* Service versions disclosed (OpenSSH, Apache)
* Network exposure identified
* Weak cookie configuration (missing Secure & HttpOnly flags)
